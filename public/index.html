📘 Passo a passo: Integrar com Bling ▼ Expandir
1
Acesse o painel do Bling
Entre em bling.com.br → Configurações → API → Gerenciar aplicativos
2
Crie um novo aplicativo
Clique em "Novo aplicativo" · Nome: iSafe CRM · No campo Redirect URI, coloque exatamente:
https://crm.lojaisafe.com.br/api/bling/oauth/callback
3
Configure os escopos de acesso
Marque obrigatoriamente: contatos · pedidos de venda · hooks (webhooks)
4
Cole as credenciais nas Configurações
Copie o Client ID e Client Secret gerados pelo Bling e cole na aba Configurações → Bling ERP deste painel. Não é necessário editar nenhum arquivo.
5
Autorize o acesso com um clique
Após salvar as credenciais, clique em "🔑 Conectar ao Bling" na aba Configurações. Você será redirecionado para o Bling para autorizar — o token é salvo automaticamente.
6
Sincronize o histórico
Na aba Configurações → Sistema, clique em "🔄 Sincronizar" para importar pedidos e contatos dos últimos 90 dias. O webhook é ativado automaticamente na inicialização do sistema.
🛍️ Passo a passo: Integrar com Shopify ▼ Expandir
1
Acesse seu painel Shopify
Entre em admin.shopify.com → Configurações → Apps e canais de vendas → Desenvolver apps
2
Crie um app privado
Clique em "Criar um app" → Nome: iSafe CRM. Configure as permissões da API de administração:
Leitura: orders, customers, products
3
Cole as credenciais nas Configurações
Após instalar o app, copie o Admin API access token e o nome da loja (ex: lojaisafe.myshopify.com) e cole na aba Configurações → Shopify deste painel. Não é necessário editar nenhum arquivo.
4
Configure o webhook no Shopify
Shopify → Configurações → Notificações → Webhooks. Crie um webhook para o evento orders/paid:
https://crm.lojaisafe.com.br/webhook/shopify
5
Sincronize o histórico
Na aba Configurações → Shopify, clique em "🔄 Sync histórico" para importar todos os pedidos dos últimos 365 dias.
🏪 Passo a passo: Integrar com Bagy ▼ Expandir
1
Acesse o painel da Bagy
Entre em app.bagy.com.br → Configurações → Integrações → API
2
Gere sua API Key
Clique em "Gerar Token de Acesso" e copie o token. O Store ID está visível na URL do painel: app.bagy.com.br/lojas/SEU_ID
3
Cole as credenciais nas Configurações
Cole o Store ID e a API Key na aba Configurações → Bagy deste painel. Clique em "💾 Salvar". Não é necessário editar nenhum arquivo.
4
Configure o webhook da Bagy
Bagy → Configurações → Webhooks → Novo webhook. Evento: Pedido aprovado. URL:
https://crm.lojaisafe.com.br/webhook/bagy
5
Sincronize o histórico
Na aba Configurações → Bagy, clique em "🔄 Sync histórico" para importar todos os pedidos dos últimos 365 dias.
